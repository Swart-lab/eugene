EuGèneHom web site :
  EuGene'Hom : a generic similarity-based gene finder using multiple homologous sequences.

Signal plugins :
  NetStart : predictions of translation start in vertebrate and Arabidopsis thaliana nucleotide sequences.
  ATGpr : a program for identifying the initiation codons in cDNA sequences.
  NetGene2 : predictions of splice sites in human, C. elegans and A. thaliana DNA.
  SplicePredictor : a method to identify potential splice sites in (plant) pre-mRNA by sequence inspection using Bayesian statistical models.
  GeneSplicer : a fast, flexible system for detecting splice sites in the genomic DNA of various eukaryotes.

Content plugins :
  Blast : Translated query vs. protein database (blastx), translated query vs. translated database (tblastx)...
  Sim4 : addresses the problem of efficiently aligning a transcribed and spliced DNA sequence (mRNA, EST) with a genomic sequence containing that gene, allowing for introns in the genomic sequence (taking into account consensus splice signals) and a relatively small number of sequencing errors.
  RepeatMasker : screens DNA sequences in fasta format against a library of repetitive elements and returns a masked query sequence ready for database searches as well as a table annotating the masked regions.
