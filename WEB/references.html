Publications/projects using EuGene :
† CATMA. The CATMA (Complete Arabidopsis Transcriptome MicroArray) has been designed using EuGene predictions on A. thaliana genome.
† Medicago truncatula genome annotation has been done using EuGene. See also the IMGAG (International Medicago Genome Annotation Group) pages for more information.
† TAIR 6 includes hundreds of new genes that have been predicted by EuGene and validated using RACE at TIGR.
